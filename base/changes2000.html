pTeX新着情報とホームページ更新情報(2000年分)

過去の変更： 1999年／ 1998年／ 1997年／ 1996年／ 1995年

2000/11/13

  • pTeXに対するパッチを公開します。
    ptex 2.1.10を --version オプションで起動したときに、表示されるバージョン番号が 2.1.9のママなのを修正します。
  • PLtoTFに対するパッチを公開します。
    CHARSINTYPEに全角空白文字を指定したとき、誤ったJFMが作られてしまうバグを 修正します。
  • pTeXライブラリを更新しました （ptex-texmf-1.8.tar.gz）。
    • pLaTeX を LaTeX <2000/06/01> に対応。
    • 縦組クラス（t{article,book,report}）で、文頭の全角開き括弧類がさがるのを修正。

2000/11/10

  • pTeXに対するパッチを公開します。
    ある種のjfm（glueパターン数 > 文字タイプ数）をロードできないバグを修正します。

2000/09/26

  • 10/6に研究集会「非欧文言語におけるTeX」が開かれます。

  • mendexkをバージョンアップしました。
    Shift-JIS処理時に、ページ番号に付加するコマンドに2バイト文字を 使用された場合に動作がおかしくなるのを修正してあります。

  • dvipsをバージョンアップしました。つぎの２点を修正してあります。
    • Type1フォントを使用した場合に、英字がずれる場合がある。
    • -zオプションを付けた場合に、PSファイル中の改行がおかしくなる。
    パッチファイルは、teTeX対応版（dvipsk-jpatch-p1.5c.tar.gz）と、通常版（dvipsk-jpatch-p1.4e.tar.gz）とがあります。

2000/05/22

  • mendexkをバージョンアップしました。
    主索引項目の下に、2つ以上の副索引項目があるとき、 副索引間のコマンド文字列がおかしくなる場合があるのを修正してあります。

2000/03/01

  • dvipskのパッチ（1.5b）を正式公開します。
    p1.5a の Makefile.in の修正（1行目の行頭に # を追加）だけです。

2000/02/25

  • dvipskのパッチ（1.5a）を正式公開します。β版からの修正はありません。

2000/02/18

  • dvipskのパッチ（1.5a）をβ公開します。以下の点を修正してあります。 
    ・Omegaモードでのコンパイルができるように修正。
・各種設定ファイルがインストールされるように Makefile を修正。
・インストール方法を変更。

2000/02/02

  • mendexkをバージョンアップしました。
    ひとつの項目に対する、複数個のページ番号の並び順がおかしくなるのを修正 してあります。

2000/02/01

  • dvipskのパッチ（1.4d, 1.5）を正式公開します。β版から、つぎの変更がされています。 
    ・漢字コードの範囲を94区94点から120区94点に拡張。

2000/01/25

  • dvipskのパッチ（1.4d, 1.5）をβ公開します。
    • dvipsk-jpatch-p1.4d.tar.gz
      dvips-5.78に対するパッチ。以下の修正を施してあります。 
      p1.4c -> p1.4d
・tpicにより作成された図が丸め誤差の蓄積によりずれてしまうことがある。
  東京理科大学の松田一朗さんのご協力により修正。
・HyperTeX の hyper.sty 使用時に、fixhyperbox のオプションを指定すると dvips
  が落ちる。慶応義塾大学の内山孝憲さんのご協力により修正。
・出力ファイル名が正しくならない場合がある。近畿大学の角藤亮さん、松阪大学の
  奥村晴彦さんのご協力により修正。
・元プログラムである dvipsk がGPL(GNU General Public License)に従っているため、
  本パッチも配布条件をGPLに従うように変更。
・pstricksの\pstextpath中で日本語を使用するとPSエラーが出る。東京理科大学の
  松田一朗さんによる対処方法を PSTricks.patch に記載。
    • dvipsk-jpatch-p1.5.tar.gz
      dvipsk-5.86（teTeX-1.0に付属）に対するパッチ。
