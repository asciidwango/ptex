pTeX新着情報とホームページ更新情報(2005年分)

過去の変更： 2004年／ 2003年／ 2002年／ 2001年／ 2000年／ 1999年／ 1998年／ 1997年／ 1996年／ 1995年

2005/11/2

  • ptex-src-3.1.9.tar.gzを公開します。
    • mkconf: platex の検索パスに tex/platex を入れないように修正
    • ptex: 単語の先頭で LIG/ を行うようなTFMを使用するとベースラインシフトが0でない場合に異常終了するバグを修正

  • dvipsk-jpatch-p1.7a.tar.gzを公開します。
    • 和文フォントのデフォルト文字幅を修正

2005/5/16

  • mendexk2.6c.tar.gzを公開します。
    • SJISモード時の出力の文字化けを修正。
    • 出力メッセージ中のaccept、rejectの数値を修正。
    • teTeXのconfigureで指定されたコンパイラを使用するようにMakefile.inを修正。

2005/3/24

  • mendexk2.6b.tar.gzを公開します。
    • 読み展開時のエラー表示の漢字コードを修正。
    • 非漢字部の範囲を厳密化。

2005/3/14

  • ptex-src-3.1.8.1.tar.gzを公開します。
    • Makefile.in: make install の最後に mktexlsr の実行を追加
    • Makefile.in, INSTALL.txt: teTeX-3.0のTEXMFディレクトリの変更に伴い修正 (ありがとうございます、角藤さん)。

  • dvipsk-jpatch-p1.7.tar.gzを公開します。
    • teTeX 3.0に含まれているdvipsk-5.95aに対応。
    • デフォルトでUnicodeへ対応。

  • mendexk2.6a.tar.gzを公開します。
    • 不正な索引行の読み込みを禁止するよう修正。
    • インストールにconfigureを使用するように変更。
    • 漢字コードオプションを追加。

2005/1/4

  • ptex-texmf-2.3.tar.gzを公開します。
    • plfonts.dtx: \fontfamily中のフラグを修正 (ありがとうございます、角藤さん)。
    • plvers.dtx: バージョン番号更新(platex<2005/01/04>)
