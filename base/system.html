TeXシステムで使うプログラム

現在のTeXシステムは、かなり大きなシステムになっていますが、 TeXシステムの中核をなすプログラム群は、大きく分けて、 文書ファイルを処理してDVIファイルを作成するためのプログラムと、 作成されたDVIファイルを処理するためのプログラムの 二つに分類することができます。
文書ファイルを処理して、DVIファイルを作成するプログラムには、 つぎのようなものがあります。
  • TeX
    Knuth博士によるプログラム。以下のすべてのプログラムの基礎。
  • LaTeX
    Leslie Lamport氏によるTeX用マクロセット。現在は、LaTeX project termに より開発されたLaTeX2eが主流になっている。
  • pTeX
    アスキーにより、日本語に対応したTeXの上位互換プログラム。 縦組や禁則処理などの機能も追加されている。
  • pLaTeX
    アスキーにより、日本語に対応したLaTeXの上位互換マクロセット。 pTeX上で動作する。現在、LaTeX2eに対応してpLaTeX2eとなっている。
