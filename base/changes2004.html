pTeX新着情報とホームページ更新情報(2004年分)

過去の変更： 2003年／ 2002年／ 2001年／ 2000年／ 1999年／ 1998年／ 1997年／ 1996年／ 1995年

2004/12/15

  • ptex-src-3.1.8.tar.gzを公開します。
    • ptex: コントロールシーケンス名の処理誤りを修正。 (例：\Te^^5e^58 が \TeX として評価されない)
    • ptex: コントロールシーケンス直後の ^^形式が漢字として認識されて しまう場合があるのを修正 (例：\TeX^^83A ⇒ \TeXア) (以上、ありがとうございます、井上さん)。
    • ptex: 端末やログファイルでの改行位置のタイミングの修正 (ありがとうございます、角藤さん)。

2004/12/9

  • ptex-src-3.1.7.tar.gzを公開します。
    • ptex: ^^形式を含むコントロールシーケンス名への対応誤りを修正しました。
    • ptex: 表示不可能文字の範囲を3.1.4と同じにしました (Missing character では ^^ 形式で出力します)。
    • mkconf: TEXINPUTS.{ptex,platex}-{sjis,jis,euc}のエントリを 作成するようにしました。

2004/12/7

  • ptex-src-3.1.6.tar.gzを公開します。
    • ptex: tetex-2.99.4.20041128-beta に対応(ありがとうございます、角藤さん)。

  • ptex-src-3.1.5.tar.gzを公開します。
    • ptex: コントロールシーケンス名の、^^形式の文字と次の文字を 漢字としてみなす場合があるのを修正
    • ptex: JISエンコーディングファイルを読み込んだ後、`漢字イン' の状態が ファイルの終わりを越えて有効になってしまう場合があるのを修正 (以上、ありがとうございます、井上さん)。
    • ptex: Missing character などのメッセージでの印字不可能文字 を^^形式で出力するように修正。
    • pltotf: SJISコードのPLファイルから、TFMファイルを正しく作れない のを修正(ありがとうございます、角藤さん、青木さん)。

  • dvipsk-jpatch-p1.6a1.tar.gzを公開します。
    • $TEXMF/dvips/base/config.ps が作られないように Makefile を修正。

2004/11/25

  • 「TeX関連のサービス窓口」の プレインさんのTeX編集部の連絡先が変わりました。

2004/09/13

  • ptex-src-3.1.4.tar.gzを公開します。
    • 行末記号を漢字の第1バイトに当たる文字にした場合、行の次の文字 と共に漢字にしようとしてしまうのを修正。
    • 1バイト文字のうち、漢字の第1バイトに当たる文字のカテゴリー コードが無効な文字となってしまうのを修正。
    • ^^形式で入力された文字コードが漢字の第1バイトに当たる場合、 次の文字と共に漢字にしようとしてしまうのを修正。
    • \string で扱う文字の漢字判定を厳密化。
    • jbibtexを改行コードCRに対応。 (以上、ありがとうございます、井上さん)
    • 数式内で \char を使用した場合に1バイト文字が漢字と誤認識される バグを修正(ありがとうございます、角籐さん)。
    • 契約条文和訳を修正。

  • ptex-texmf-2.2.tar.gzを公開します。
    • verbatim環境で余計な前後空きが発生しないように \fontfamily を修正。
    • JY1,JT1以外の和文エンコーディングも使用できるように修正。
    • \part, \chapter の \@afterindentfalse を \@afterindenttrue に変更。
    • jclasses.dtx: \section, \subsection, \subsubsection の前後空きの 伸縮幅を修正。
    • LaTeX2e 2003/12/01 版への対応を確認。
    • min, goth, nmin, ngoth のtfmに余計な文字コードの定義があったので 削除。
    • 契約条文和訳を修正。

  • dvipsk-jpatch-p1.6a.tar.gzを公開します。
    • make install で config.ps, psfonts.map がインストールされるように Makefile を修正。
    • dvi2ps用のマクロでEPSファイルを読み込んだ場合にMac用改行コードが正しく 認識されない不具合を修正。

  • mendexk2.5a.tar.gzを公開します。
    • 引数の入力ファイル名から拡張子(.idx)を省略した場合にエラーが発生するバグを 修正。
