● 日本語 TeX バージョン 3.14159, p2.1.8 について
---------------------------------------------------------------------------

 * 日本語 TeX バージョン 3.14159 p2.1.8（以下 pTeX）は、TeX を改造して本格
   的な商業出版に耐える TeXを開発することを目標にしています。 このバージョ
   ン（3.14159, p2.1.8）は、日本語 TeX バージョン 2.99 j1.7に「縦組み機能」
   を追加した「日本語 TeX バージョン 2.99 j1.7 p1.0.9G」と同等の機能を持っ
   ています。

 * 以前は横組み専用の「日本語TeX」と縦組みにも対応した「pTeX」をリリースし
   ていましたが、現在は pTeX のみをリリースしています。 したがって日本語化
   のバグフィックスはすべて pTeX に対してのみ行ない、横組のみの日本語 TeX
   に対しては行ないませんので、ご了承ください。

 * pTeX は、WEB のチェンジファイルを使って実現されています。現在は、 バー
   ジョン 3.14159 用のチェンジファイルとなっています。

 * pTeX は、現在、Linux (Intel Pentium Pro)で開発、動作を確認しています。
   EUC, シフトJIS のどちらか、および JIS コードのファイルを処理できます。

● 必要なファイル
---------------------------------------------------------------------------
pTeX をインストールするには、つぎのファイルが必要です。

 * ptex218.tar.gz            (pTeX ソースファイル)
 * web-7.2.tar.gz            (TeX ソースファイル)
 * web2c-7.2.tar.gz          (Web2c ソースファイル)
 * texmflib-7.3.tar.gz       (TeX ライブラリファイル)
 * plib-1.4-{euc,jis,sj}.tar.gz  (pTeX ライブラリファイル)

● インストール
---------------------------------------------------------------------------

              ※※※※※※※※※※※※※※※※※※※※※
              インストール作業をする前に、旧バージョンの
              TeXシステムのバックアップを取ってください。
              ※※※※※※※※※※※※※※※※※※※※※

pTeX のインストールは、つぎの順序で行います。

 * TeX のインストール
    - ライブラリのインストール
    - プログラムのインストール
 * pTeX のインストール
    - ライブラリのインストール
    - プログラムのインストール

--------------------------------------
・TeX のインストール
--------------------------------------
ここでは、TeX のインストール方法について簡単に説明をします。詳細について
は、web2c-7.2.tar.gz に付属の INSTALL ファイルを参照してください。

TeX のインストールは、以下の手順で行います。

1. texmflib-7.3.tar.gz ファイルを展開します。
   /usr/local/share ディレクトリの下に置く場合は、つぎのようにします。

      tar zxvf texmflib-7.3.tar.gz -C /usr/local/share

2. web-7.2.tar.gz ファイルと web2c-7.2.tar.gz ファイルを展開します。
   /usr/src ディレクトリで作業をする場合は、つぎのようにします。
   実行後、/usr/src ディレクトリに web2c-7.2 ディレクトリが作成されます。

      tar zxvf web-7.2.tar.gz -C /usr/src
      tar zxvf web2c-7.2.tar.gz -C /usr/src

3. ソースファイルを展開したディレクトリに移動し、configure コマンドを実行
   します。

      cd /usr/src/web2c-7.2
      ./configure

4. make を実行します。

5. make install を実行します。

--------------------------------------
・pTeX のインストール
--------------------------------------
1. plib-1.4-{euc,jis,sj}.tar.gz のいずれかのファイルを展開します。
   /usr/local/share ディレクトリの下に置く場合は、つぎのようにします。

      tar zxvf plib-1.4-{euc,jis,sj}.tar.gz -C /usr/local/share

2. ptex218.tar.gz ファイルを TeX のソースディレクトリの web2c の下に展開
   します。次の例は、/usr/src/web2c-7.2/web2c の下に展開をします。

      tar zxvf ptex218.tar.gz -C /usr/src/web2c-7.2/web2c

3. ptex218 ディレクトリに移動し、configure コマンドを実行します。
   このとき、端末に出力する漢字コードを指定してください。

      cd /usr/src/web2c-7.2/web2c/ptex218
      ./configure EUC     (EUCコードの場合)
      ./configure JIS     (JISコードの場合)
      ./configure SJIS    (シフトJISコードの場合)

4. make を実行します。

5. make install を実行します。

---------------------------------------------------------------------------
自動生成される ascgrp, ascii10, ascii36 の pk ファイルを
$TEXMF/fonts/pk/<モード>/ascgrp の下に置くようにする場合は、
$TEXMF/fontname/special.map につぎの二行を加えます。

    ascgrp ascgrp
    ascii  ascgrp
